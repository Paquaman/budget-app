# Budget Ramsey 💰

Application de gestion budgétaire inspirée des principes de **Dave Ramsey** — budget base zéro, méthode boule de neige, et les 7 étapes vers la liberté financière.

## Fonctionnalités

- **Tableau de bord** — Vue d'ensemble avec ratios budgétaires en temps réel
- **Revenus** — Revenu principal + sources additionnelles
- **Dépenses** — Catégorisées avec pourcentages recommandés
- **Investissements** — REER, CELI, fonds d'urgence avec suivi de l'objectif 15%
- **Dettes** — Méthode boule de neige (tri par solde croissant)
- **Ratios** — Logement, dépenses, investissements, dettes, dons

## Ratios recommandés (Dave Ramsey)

| Catégorie | Recommandé |
|---|---|
| Logement | 25–35% |
| Alimentation | 10–15% |
| Transport | 10–15% |
| Investissements | 15% |
| Dons | 10% |
| Dettes | 0% (objectif) |

## Installation

```bash
npm install
npm run dev
```

Ouvrez [http://localhost:5173](http://localhost:5173)
